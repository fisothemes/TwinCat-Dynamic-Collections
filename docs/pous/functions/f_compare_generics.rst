F_Compare_Generics (Function)
=============================

Compares generic value A to generic value B and returns the evaluation.

**Returns:** :ref:`T_Comparison <t_comparison>`

**Inputs**

.. list-table::
   :header-rows: 1
   :widths: 25 20 55

   * - Name
     - Type
     - Description
   * - ``A, B``
     - :ref:`T_Generic <t_generic>`
     -


**Outputs**

.. list-table::
   :header-rows: 1
   :widths: 25 20 55

   * - Name
     - Type
     - Description
   * - ``bUndefined``
     - ``BOOL``
     - If TRUE, values were compared but truthfulness of comparison cannot be verified.
